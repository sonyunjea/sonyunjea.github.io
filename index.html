AI 쓰레기통, 완벽을 향한 도전

'음식물 묻은 플라스틱' 문제를 해결하여 재활용의 미래를 여는 기술 고도화 프로젝트를 소개합니다.

시작부터 마주한 기술적 난관

기능 추가를 위해 초기 계획으로 수집한 50장의 사진. 이 작은 데이터셋이 어떻게 프로젝트의 발목을 잡을 수 있는지, 두 가지 치명적인 문제점을 발견했습니다.

문제점 1: 과적합(Overfitting) 현상

AI가 학습 데이터만 완벽하게 암기하고, 처음 보는 실제 쓰레기는 전혀 인식하지 못하는 문제입니다. 이는 훈련 성적은 100점이지만 실전 시험은 0점인 것과 같습니다. 적은 데이터 양은 모델이 편협한 시각을 갖게 하여 일반화 성능을 심각하게 저해합니다.

50
초기 학습 이미지

무한에 가까운 실제 환경의 변수를 반영하기엔 절대적으로 부족한 양입니다.

문제점 2: 분류 클래스의 모호성

서로 다른 특징을 가진 대상을 하나의 클래스로 묶어 가르치면, AI는 명확한 분류 기준을 세우지 못하고 혼란에 빠집니다. 이는 전체 시스템의 성능 저하로 이어집니다.

AI 혼란 야기 (Confusion!)

데이터 중심의 체계적인 해결 방안

문제 해결을 위해 우리는 머신러닝 개발 방법론에 입각한 3단계의 핵심 전략을 수립했습니다.

🖼️✨

데이터 증강

50장의 원본을 회전, 반전, 확대/축소하여 500장 이상의 데이터셋으로 확장, 모델의 강건함(Robustness)을 확보합니다.

🤖🧠

전이 학습

이미 똑똑한 사전 학습 모델(YOLO v8)을 기반으로, 우리에게 필요한 '오염된 플라스틱'만 효율적으로 추가 학습시킵니다.

🏷️🔄

클래스 재설계

AI가 혼란 없이 명확한 기준으로 학습할 수 있도록, 독립적인 클래스를 신설하여 분류 체계를 재정의합니다.

새로운 청사진: 명확해진 분류 기준

복잡한 '인식'과 단순한 '행동'을 분리하는 효과적인 설계를 통해 시스템의 안정성과 확장성을 모두 높입니다.

BEFORE: 모호한 3개 클래스

  • 1. 일반쓰레기/종이
  • 2. 플라스틱/병
  • 3. 캔

AFTER: 명확한 4개 클래스

  • 1. 일반쓰레기
  • 2. 재활용 가능
  • 3. 캔
  • 4. 오염된 플라스틱 (신설)

성공을 위한 5단계 세부 수행 계획

데이터 수집부터 최종 테스트까지, 체계적인 로드맵을 따라 프로젝트를 완수합니다.

1단계: 데이터 수집 및 가이드라인 수립

고품질 원본 데이터 50장 이상 확보

2단계: 데이터 전처리 및 증강

학습 데이터셋 500장 이상으로 확장

3단계: 모델 재설계 및 학습

YOLO v8 기반 전이 학습 수행

4단계: 성능 평가 및 최적화

mAP 지표로 성능 측정 및 최적화

5단계: 시스템 통합 및 최종 테스트

엣지 디바이스에 배포 및 실시간 검증

프로젝트 기대 효과

본 프로젝트를 통해 AI 쓰레기통은 한 단계 더 진보하여, 더 높은 정확도와 안정성, 그리고 확장성을 갖추게 됩니다.

예상 분류 정확도

체계적인 개선을 통해 기존 모델 대비 분류 정확도를 대폭 향상시켜 재활용 효율을 극대화할 수 있습니다.

🎯

일반화 성능 및 안정성 확보

어떤 쓰레기가 들어와도 일관되고 안정적인 성능을 보장합니다.

📈

정확도 및 정밀도 향상

오분류 오류를 최소화하여 재활용 자원의 손실을 막습니다.

🔧

유지보수 및 확장성 강화
